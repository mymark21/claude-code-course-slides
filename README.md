# Claude Code 课程演示

日系侘寂风格的 Claude Code 课程演示文稿。

## 🎨 设计特点

- 温暖的米色/奶油色背景
- 优雅的中文衬线字体（思源宋体）+ 西文衬线字体
- 精致的框线装饰（四角边框、顶部装饰线、圆点）
- 克制的配色（深红、暖棕、鼠尾草绿）
- 细腻的边框和分隔线

## 📖 内容大纲

- **PART 1: 基础** - 从文件夹开始、模型选择、两种模式、上下文管理
- **PART 2: 工作流心法** - Prompt 写作、对话管理、输出审查
- **PART 3: 进阶** - Ghostty 终端、自举、Agent 蜂群、Skill 体系、MCP
- **PART 4: Demo** - 创建网站、文章写作、SEO 工作流、RPA 自动化

## 🎮 使用方法

### 导航
- 方向键 / 空格键 / 滚轮：切换幻灯片
- Home / End：跳转到首页/末页
- 点击右侧导航点：直接跳转

### 编辑模式
- 悬停左上角或按 `E` 键：进入编辑模式
- 点击任意文本编辑
- `Ctrl+S` / `Cmd+S`：保存（自动保存到 localStorage）

## 🎨 自定义配色

编辑 `claude-code-course.html` 文件中的 CSS 变量：

```css
:root {
    --bg-cream: #faf9f7;
    --text-primary: #1a1a1a;
    --accent-crimson: #c41e3a;
    --accent-warm: #d4a574;
    --accent-sage: #8b9d83;
    /* ... */
}
```
